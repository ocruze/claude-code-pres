Démarrer en 5 min

  1. Installer Claude Code : npm i -g @anthropic-ai/claude-code
  2. Lancer dans ton projet : claude
  3. Créer un CLAUDE.md avec tes conventions clés
  4. Installer le plugin Figma : /plugin install figma
  5. Coller une URL de nœud Figma dans le prompt !
